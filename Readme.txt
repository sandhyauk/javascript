Step 1: Install Required Tools
Go to Company Portal and install:

* Visual Studio Code
* Node.js (Latest)

---

Step 2: Install Git

Open Command Prompt and run:

winget install --id Git.Git -e --source winget

---

Step 3: Goto users folder

C:\Users\san8577---{} replace this with your name

---

Step 4: Restart

Restart your computer (or at least close and reopen vs code)

---

Step 5: Verify Installation

Run the following commands:

node -v
npm -v
git --version

Open terminal
type cd C:\Users\san8577\PlaywrightRepos

Whenever updates are made, run:
